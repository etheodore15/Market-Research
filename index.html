Quick Start

  1. Clone the repo: 
    git clone https://github.com/etheodore15/Market-Research.git
cd Market-Research
  2. Install dependencies: 
    npm install
  3. Add your API key: 
    cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY
  4. Run the agent: 
    npm start

How It Works

  1. The agent runs 6 sequential research modules using Claude Opus with live web search enabled.
  2. Each module streams results to the terminal in real time as Claude searches the web and writes analysis.
  3. On completion, the full report is compiled into a timestamped markdown file.
  4. A desktop notification alerts you when the research is done.
